# 📊 Mutual Fund Analytics Capstone Project

## 📌 Project Overview

The **Mutual Fund Analytics Capstone Project** is an end-to-end data analytics solution developed to analyze the Indian mutual fund industry using Python, SQL, SQLite, and Power BI.

The project covers the complete analytics workflow, including data ingestion, data cleaning, exploratory data analysis (EDA), fund performance analytics, and interactive dashboard development. It provides meaningful insights into mutual fund performance, investor behavior, portfolio allocation, SIP trends, and benchmark comparisons.

---

## 🎯 Project Objectives

- Build an automated ETL pipeline for mutual fund datasets.
- Clean, validate, and standardize raw financial data.
- Design a relational SQLite database.
- Perform SQL-based analytical queries.
- Conduct Exploratory Data Analysis (EDA) using Python.
- Compute key mutual fund performance metrics.
- Develop an interactive Power BI dashboard.
- Prepare a professional project report and presentation.

# 🔍 Key Insights

- Mutual fund NAVs demonstrated long-term growth with periodic market corrections.
- Monthly SIP inflows increased steadily during the analysis period.
- Equity mutual funds attracted higher investor participation than debt and hybrid funds.
- Retail investor folio counts continued to grow across the industry.
- Financial Services and Information Technology sectors accounted for significant portfolio allocations.
- Risk and return exhibited a positive relationship across most equity schemes.
- Benchmark comparisons highlighted differences in fund performance.
- Expense ratios varied across schemes and influenced long-term performance consistency.
